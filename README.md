## 🌟 Overview
**Cyber Scam AI** is a high-performance security dashboard designed to protect Web3 investors from malicious actors. By combining static bytecode analysis with behavioral AI heuristics, the platform identifies high-risk vulnerabilities—such as honeypots and hidden mint functions—before they result in capital loss.

## 🚀 Key Features
- **AI-Powered Risk Scoring:** Real-time analysis of contract logic to generate a 0-100 risk percentage.
- **Deep Scan Engine:** Detects 5+ critical vulnerability types including Liquidity Drain and Tax Manipulation.
- **Professional Dashboard:** A sleek, 2026-grade UI designed for rapid due diligence.
- **Simulated Behavioral Checks:** Identifies "soft" red flags like wallet clustering and unrenounced ownership.

## 🛠️ Tech Stack
- **Frontend:** [Streamlit](https://streamlit.io/) (Python-based Web Framework)
- **Design:** Custom CSS3 with Dark-Mode optimization.
- **Blockchain Logic:** Web3.py for real-time contract interaction.
- **AI Model:** Heuristic-based risk engine (Evolving to Neural Network in v2.0).

## 📁 File Structure
- `app.py`: The main entry point for the UI and navigation.
- `engine.py`: The core AI logic and vulnerability detection engine.
- `theme.css`: Custom professional styling and neon-branding.
- `requirements.txt`: Necessary Python dependencies.
